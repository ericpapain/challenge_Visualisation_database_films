Challenge – Visualisation d’une base de données de films

Sujet

Vous disposez d’un fichier contenant une base de données de films (movies.csv). Votre mission est de produire une plateforme permettant de visualiser au mieux l’intégralité de cette base de données.  

Livrables attendus

Vous devez fournir :

1\.	Un document de synthèse de 2 pages maximum, contenant :

•	votre méthode ;

•	les fonctionnalités principales, avec au moins deux figures ;

•	une discussion des limites et améliorations possibles ;

•	votre usage des outils d’IA dans ce challenge.

2\.	Un code reproductible et documenté sous Github, qui reconstruit la plateforme à partir du fichier movies.csv.

Nous valorisons particulièrement :

•	la méthodologie ;

•	la reproductibilité du code et sa documentation ;

•	l’ergonomie de la plateforme ;

•	la multimodalité des données représentées (texte, image, labels) ;

•	l’analyse critique.

Usage d’outils d’IA

L’usage d’outils d’IA est vivement encouragé.

Vous devez toutefois inclure dans votre note de synthèse quel outil vous avez utilisé, pour quelles tâches, et quelles vérifications vous avez réalisées vous-même.

Vous devez être capable d’expliquer en entretien chaque bloc de code important, chaque choix méthodologique, chaque fonctionnalité présentée et chaque limite discutée.

Temps indicatif : 2 à 3 heures.
